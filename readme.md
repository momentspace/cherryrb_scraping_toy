# cherry.rb scraping toy

## Installation
1. git clone

  `git clone https://github.com/kytiken/cherryrb_scraping_toy`

2. change directory

  `cd cherryrb_scraping_toy`

3. bundle install

  `bundle install --path vendor/bundle --jobs 4`

## How to play

1. execute guard

  `bundle exec guard`


2. please change app.rb
